# Katkıda Bulunma

Free Stoks Photo projesine katkılarınızı bekliyoruz.

## Geliştirme ortamı

```bash
git clone https://github.com/EfeOkty/Free-Stoks-Photo.git
cd Free-Stoks-Photo
npm install
npm run dev
```

## Değişiklik gönderme

1. Bu repoyu fork edin.
2. Yeni bir branch açın: `git checkout -b feature/amaciniz`.
3. Değişikliklerinizi commit edin: `git commit -m "Kısa açıklama"`.
4. Branch'i push edin: `git push origin feature/amaciniz`.
5. GitHub üzerinden Pull Request açın.

## Davranış kuralları

Saygılı ve yapıcı iletişim bekliyoruz. Proje MIT lisansı altındadır; katkılarınız da aynı lisansla kabul edilir.
